Screen

Screen is a full-screen window manager that 
multiplexes a physical terminal between 
several processes, typically interactive 
shells.

.screenrc

vbell on
nethack on 
vbell_msg ""
autodetach on
startup_message off
pow_detach_msg "Screen session of \$LOGNAME \$:cr:\$:nl:ended."
shell -$SHELL
msgminwait 3

moving around

  • ctrl-a 0-9
  • ctrl-a n
  • ctrl-a p
  • ctrl-a ctrl-a
  • ctrl-a ' / name or number
  • ctrl-a " / select using j/k & enter

other commands

  • new window: ctrl-a c
  • name window: ctrl-a shift-a
  • scroll: ctrl-a esc / movement (j, k, ctrl-b, ctrl-f)
  • kill window: ctl-a shift-k
  • see all commands: ctrl-a ?

vim

Vim is an advanced text editor that seeks 
to provide the power of the de-facto Unix 
editor 'Vi', with a more complete feature 
set.

modes

  • Normal mode: esc
  • Visual mode: v, ctl-v, shift-v
  • Insert mode: i, a, o, etc.
  • Command-line mode: :

moving around in normal mode

  • j, k, l, m - up, down, left, right
  • ctrl-b, ctrl-f - page up, page down
  • ctrl-e, ctrl-y - scroll down, scroll up
  • gg, G - first line, last line
  • ^, $ - beginning of line, end of line
  • :100 - go to line 100
  • w, b - forward one word, back one word

copy, paste, & delete

  • yy - copy (yank)
  • p - paste (put)
  • dw - delete word
  • d$ - delete to end of line
  • dd - delete entire line

undo & redo

  • u - undo last change
  • ctrl-r - redo

search & search/replace

  • /
  • n, p - find next, find previous
  • :s/search/replace/g - current line
  • :%s/search/replace/g - entire file

open file

  • :e path/to/file
  • :sp path/to/file - horizontal split file
  • :vsp path/to/file - vertical split file

save & quit

  • :w - save
  • :q - quit
  • :wq - save & quit
  • :q! - quit without saving
